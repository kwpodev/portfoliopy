JOGO: ADIVINHE O NÚMERO (INTERFACE GRÁFICA)

Descrição:
-----------
Este é um jogo simples desenvolvido em Python usando a biblioteca Tkinter. O objetivo do jogador é adivinhar um número aleatório entre 1 e 10. A cada tentativa, o programa fornece dicas se o palpite foi muito alto ou muito baixo. O jogo termina quando o jogador acerta o número.

Como Jogar:
-----------
1. Execute o programa em um ambiente com suporte a interfaces gráficas (por exemplo, o IDLE ou diretamente no terminal com suporte ao Tkinter).
2. Uma janela será aberta com instruções.
3. Digite um número entre 1 e 10 no campo de entrada.
4. Clique em "Verificar".
5. O programa informará se o número é muito alto, muito baixo ou correto.
6. O número de tentativas será exibido ao final.

Requisitos:
-----------
- Python 3.x
- Biblioteca Tkinter (já inclusa por padrão na maioria das distribuições Python)

Execução:
---------
Para executar o programa, utilize o seguinte comando no terminal:

python nome_do_arquivo.py

Substitua `nome_do_arquivo.py` pelo nome do arquivo que contém o código.
